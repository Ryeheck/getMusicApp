 Music & Lyrics Downloader

 Приложение для удобного поиска и скачивания музыки и синхронизированных текстов песен (.lrc).

 Как пользоваться

 Укажите папку сохранения: Введите путь в поле "Enter folder...".
 Если оставить поле пустым, файлы сохранятся в стандартную папку "Музыка".
 Вставьте ссылку: Скопируйте ссылку на YouTube видео или плейлист в поле "Enter url...".
 Запустите работу: Нажмите Music download(s) для скачивания аудио.
 Нажмите Lyrics download(s), если вам нужны только тексты песен.
 Используйте кнопку Playlist для обработки целых списков воспроизведения.

##############################################################################################

A simple app for searching and downloading music and synchronized lyrics (.lrc).

How to use

Set the download folder: Enter the path in the "Enter folder..." field.
If left empty, files will be saved to your default "Music" folder.
Paste the link: Copy a YouTube video or playlist link into the "Enter url..." field.
Start downloading: Click Music download(s) to download audio files.
Click Lyrics download(s) if you only need the lyrics.
Use the Playlist button for processing entire playlists.
